You have subjects            [{ name: "Math" }, ...]
You have teachers            [{ name: "Ivanov Alex" }, ...]
You have grades              [{ level: 11, letter: "A" }, ...]
You have groups by subject   [{ subjectID: 1, name: "Group 1" }, ...]
You have rooms               [{ number: 123 }, ...]

Моковые данные сверху это просто для наглядности, можно менять их как угодно для удобства. 
Также необходимо эмулировать получение данных с сервера с небольшой задержкой. 
Данные которые сверху это предметы, учителя, классы, подгруппы, и аудитории в школе.
_____________________________________________________________________

Необходимо создать таблицу (по горизонтали классы по вертикали дни недели) в которой будут 
отображаться информация по распределенным предметам внутри школы по кабинетам.

Предметы в разрезе класса (тоесть только по вертикали) можно перемещать Drag-N-Drop'ом. 
При перетаскивании елемента необходимо подсвечивать весь столбец, или затемнять остальные как вариант. 
Дропать предмет на предмет можно только в случае если это свободное место для второй подгруппы! 
Иначе ни в каких случаях дропать предмет на предмет нельзя, только в свободную ячейку.

Учителя в школе преподают не по одному предмету а от двух до пяти. 
Например учитель Михаил который читает Алгебру еще читает Математику Геометрию 
Тригонометрию и Мат Анализ, всего пять предметов. Учитель который читает Укр Мову 
читает еще Укр Лит, тоесть два предмета. Свободных учителей в школе под конец не должно остатся, 
каждый должен чтото читать.

У каждого класса в день может быть от 5 до 7 уроков

Если предмет бьется на подгруппы то для него надо две аудитории в один момент времени, 
по скольку часть людей идет в один кабинет, а часть в другой, на информатику например.

Если учитель который читает предмет занят в этот момент времени, то ячейки будут 
перекрашиваться в красный, тоесть предметы не должны задваиватся. Тоже самое и по аудиториям, 
если задваивается аудитория то ячейка будет подсвечиватся. Например в результате 
Drag-N-Drop'а случайно получилось так что эта аудитория уже занята на первом уроке в понедельник, 
но только в другом классе, тоесть эти обе ячейки будут красные.
